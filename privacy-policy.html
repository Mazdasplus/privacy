ThePlusInspect — Privacy Policy

Developed and operated by Mazda's Plus LLC — Modesto, CA

Effective date: May 27, 2026

ThePlusInspect is a Chrome extension built for internal use by Mazda's Plus LLC service advisors. It helps annotate automotive inspection photos and upload them directly into the Tekmetric shop management system. This policy explains what data the extension accesses, how it is used, and who it is shared with.

1. Data We Access

The extension accesses the following information while in use:

  • Inspection photos — photos attached to repair orders in Tekmetric are loaded into the extension editor for annotation. These photos are sourced from Tekmetric's own servers and are not copied or stored outside of the extension's local processing.
  • Repair order information — the current Tekmetric page URL and inspection data (item names, technician ratings, and notes) are read to provide context for AI annotation.
  • Extension settings — the user's Anthropic API key and shop name are stored locally in Chrome's extension storage.

2. Data We Store

  • API key and shop name (chrome.storage.sync) — stored locally in the user's Chrome profile so settings persist across sessions. This data never leaves the user's device except to sync across the user's own Chrome profiles via Google's encrypted sync.
  • Learned arrow positions (chrome.storage.local) — the extension records where arrows were placed on past annotations for specific inspection items (e.g., "Serpentine Belt") to improve future AI suggestions. This data is stored entirely on the local device and is never transmitted anywhere.

We do not store inspection photos, customer information, vehicle data, or any personally identifiable information on any external server.

3. Data Sent to Third Parties

When a user requests AI annotation or a customer description, the inspection photo (as a base64-encoded image) and the associated technician notes are sent to the Anthropic API (api.anthropic.com) for analysis. This request is made using the API key entered by the user in the extension settings — the user controls and owns that API account.

Anthropic's data handling is governed by their own Privacy Policy. Photo data sent to the Anthropic API is used only to generate the annotation and description response and is subject to Anthropic's data retention policies.

No other third parties receive any data. The extension does not include analytics, advertising, or tracking of any kind.

4. Data We Do Not Collect

  • We do not collect customer names, contact information, or personal data.
  • We do not track usage, page views, or user behavior.
  • We do not sell, rent, or share any data with advertisers or data brokers.
  • We do not transmit inspection photos to any server other than the Anthropic API at the user's explicit request.

5. Permissions Used

  • activeTab — reads the current Tekmetric page URL to identify the repair order and shop in context.
  • tabs — used to notify open Tekmetric tabs when an extension update is available.
  • storage — stores API key, shop name, and learned arrow positions locally on the device.
  • Host access (tekmetric.com, amazonaws.com, anthropic.com) — required to inject the annotation panel into Tekmetric, fetch inspection photos from Tekmetric's photo storage, and send photos to the Anthropic API.

6. Data Security

All communication with the Anthropic API uses HTTPS encryption. The extension does not operate any servers and stores no data outside of the user's own device and Chrome profile. The API key is stored using Chrome's built-in encrypted storage.

7. Who Can Use This Extension

ThePlusInspect is an internal tool developed for use by employees and authorized staff of Mazda's Plus LLC. It is not intended for use by the general public. By installing this extension, users agree to use it only for its intended purpose of annotating automotive inspection photos within Tekmetric.

8. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected by an updated effective date at the top of this page.
